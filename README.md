# a basic 2 bit counter using JK ff
Using :
  -the fact that when j=H and k=H , Q(t+1)=Q(t)`
  -the clock of a bit is the Q of the previous bit, clk_i=Q_(i-1).
  -the main clock ,clk_0 , is achieved using 555 providing ~0.5 HZ
